# C.R.A.C.K. - Image Generation Prompt

## Project: Comprehensive Recon & Attack Creation Kit

A modular penetration testing toolkit combining CLI, GUI, and AI integration for professional security assessments.

---

## Image Generation Prompt

**Scene Description:**
A dramatic cybersecurity command center viewed from a cinematic low angle. In the foreground, a sleek dark terminal displays cascading green and cyan command outputs—1,487 penetration testing commands flowing like a digital waterfall. Behind it, a massive holographic Neo4j graph database visualization floats in mid-air, showing interconnected attack chains as glowing nodes and edges in electric blue and crimson red. The nodes pulse with energy, representing 32 attack workflows branching and connecting.

**Composition:**

- Center: A powerful workstation with three curved ultrawide monitors arranged in a semicircle
- Left monitor: Dark terminal with green monospace text showing command execution
- Center monitor: The Crackpedia GUI interface with a cyberpunk-styled knowledge graph
- Right monitor: Live engagement tracker with target enumeration data
- Above: Floating holographic attack path visualization connecting the screens
- Foreground: Subtle digital particles and data streams flowing toward the center

**Color Palette:**

- Primary: Deep matte black (#0D0D0D) and gunmetal gray (#2A2A2A)
- Accent: Electric cyan (#00FFFF), neon green (#39FF14), and blood red (#8B0000)
- Highlights: White and gold for important data points
- Ambient: Purple and blue gradient fog in the background

**Lighting:**

- Primary light source: Cool cyan glow emanating from the monitors
- Secondary: Subtle red underlighting from the desk surface
- Volumetric light rays cutting through digital fog
- Screen bloom creating a soft halo effect around each display

**Mood/Atmosphere:**
Elite, tactical, professional. The image should evoke the feeling of a master security professional's workspace—organized chaos of data streams, powerful tools at fingertips, and the quiet intensity before a penetration test engagement. Think "hacker's command bridge meets military operations center."

**Key Visual Elements:**

- Fractured/cracked visual motif subtly integrated into UI elements (referencing the name)
- Python code snippets floating as translucent overlays
- Lock icons transitioning to unlocked states
- Network topology maps with pulsing connection lines
- A stylized skull-and-crossbones made of circuit patterns (subtle watermark)

**Style References:**

- Blade Runner 2049 holographic interfaces
- Mr. Robot hacking sequences
- Ghost in the Shell digital environments
- Cyberpunk 2077 UI design language

**Technical Specifications:**

- Aspect ratio: 16:9 (ideal for hero banners)
- Style: Photorealistic with digital enhancement
- Render quality: Ultra-detailed, 8K resolution aesthetic
- Depth of field: Shallow, focusing on center monitor with bokeh on edges

**Text Overlay Space:**
Reserve the lower-right quadrant for potential logo placement and tagline: "Your Complete Penetration Testing Arsenal"
